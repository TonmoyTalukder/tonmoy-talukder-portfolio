Let's assume, there is a machine with an assignment.

Now, if the machine does fail or not, we have to find out that this is a Classification type problem of machine learning.
But, if we wanna find out after how many times the machine will fail, that is a Regression type problem of machine learning.

Now, assume there are a set of signals for that machine to do its assignment. Now if we wanna find out from those signals, why the machine is failed. Suppose there are 10000 signals. But the machine may fail for the causes created by 8 to 10 signals of them. Now, finding those 8 to 10 causes among 10000 in total by the signals is a Clustering type problem of machine learning.

So, finding out what kind of machine learning is totally depends on how I am defining the problem.
